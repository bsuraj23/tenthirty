23.Explain pass-by-value vs pass-by-reference in Python.
ans:Pass-by-Value vs Pass-by-Reference
Pass-by-Value: A copy of the actual value is passed to the function. Changes inside the function do not affect the original variable.
Pass-by-Reference: A reference (pointer) to the original object is passed. Changes inside the function affect the original object.

üîπ Python‚Äôs Behavior
Python uses a ‚Äúpass-by-object-reference‚Äù (or ‚Äúpass-by-assignment‚Äù):
Mutable objects (lists, dicts, sets) ‚Üí behave like pass-by-reference
Immutable objects (int, float, str, tuple) ‚Üí behave like pass-by-value

    return f"Hello, {name}!"

# Call the function using the new variable
